# Ex-formation

### 模块划分

- 初始化 init
  - 系统信息 system
  - 用户标识 user
- 本地存储 storage
- 全局信息存储 info
  - 全局变量
  - 计算存储对象
- 埋点采集（autoTrack）
  - 生命周期钩子（事件属性）
  - 点击事件
- 渠道来源 utm
- 分享模块 share
- 工具方法模块
  - log 日志
- 数据发送
- 全局 API

每个模块内完成各自小程序处理逻辑的实现
每个单独模块间保证不出现彼此间循环引用的情况
插件的实现（新增 api，新增变量，可以依赖 SDK 而运行），可单独集成，可以直接打包入 SDK
